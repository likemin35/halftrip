# Screen Flow

## App Flow

1. Login
2. Home
3. New Trip Application
4. Region Selection
5. Trip Detail
6. Place Info
7. Planner
8. Evidence Upload
9. Settlement
10. Refund Usage

## Key Navigation

- Login -> Home
- Home -> New Trip Form
- New Trip Form -> Region Selection
- Region Selection -> Trip Detail
- Home -> Existing Trip List
- Trip Detail -> Place Info / Planner / Evidence / Settlement / Refund Usage
- Home -> Settings

- 숙박확인서 양식은 공통 템플릿 PDF 기준
